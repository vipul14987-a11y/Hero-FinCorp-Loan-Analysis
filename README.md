# Hero FinCorp Loan Portfolio Analysis

## Objective
Analyze loan performance, customer risk factors, defaults, recovery behavior, and operational efficiency using multiple relational datasets.

## Datasets Used
- Customers
- Loans
- Applications
- Transactions
- Defaults
- Branches (aggregate level)

## Approach
1. Data quality checks and validation
2. Feature engineering to create a loan-level analytical dataset
3. Descriptive and risk-based analysis
4. Correlation and time-based trend analysis
5. Branch and regional performance evaluation

## Key Insights
- Credit score is a strong predictor of loan default.
- Lower income bands exhibit higher default rates.
- Default rates remain broadly stable over time with occasional spikes.
- Certain regions contribute disproportionately to delinquent loans.
- Branches with longer processing times show higher delinquency.
